# Vibe Channel

Vibe Channel is a video-sharing platform built entirely using React.js and RapidAPI, allowing users to browse and watch videos without the need for backend servers or authentication.

## Features

- **Video Search**: Users can search for videos across a wide range of categories.
- **Video Viewing**: Users can watch videos directly within the platform.
- **Responsive Design**: The platform is designed to work seamlessly across various devices and screen sizes.

## Technologies Used

- **Frontend**:  React.js
- **API Integration**: RapidAPI for fetching video data
- **Other Tools**: Git for version control

## Getting Started

To get started with Vibe Channel, follow these steps:

1. Clone this repository to your local machine.
2. Install dependencies using `npm install`.
3. Obtain an API key from RapidAPI for accessing video data.
4. Set up the API key in the appropriate configuration file or environment variable.
5. Start the React development server using `npm start`.
6. Access the application in your web browser at `http://localhost:3000`.
